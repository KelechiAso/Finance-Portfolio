My Learning Journey: Terminal Value Calculation

I've learned that the Terminal Value (TV) is a crucial part of a DCF model that represents a company's value beyond the explicit forecast period. Since no one can accurately forecast a companyâ€™s cash flows forever, this is the way I account for all future cash flows after the first 5-10 years. It often accounts for a huge portion (60-80%) of the total DCF valuation, which is why understanding its calculation is so critical.

Method 1: The Perpetuity Growth Method

This is the first method I learned for calculating Terminal Value. It assumes that a company's Free Cash Flow will grow at a constant, sustainable rate forever. I think of it as a modified version of the perpetuity formula. It takes the final year's projected cash flow, grows it by a conservative rate, and then divides it by the difference between the discount rate and the growth rate.

The formula I use is:

$$Terminal\ Value = \frac{FCFF_n \times (1+g)}{WACC-g}$$

Here's how I break down the components:

  • $FCFF_n$ is the Free Cash Flow in the final year of my explicit forecast period.
  • $g$ is the Perpetuity Growth Rate, which I assume to be a conservative long-term growth rate, typically not exceeding the long-term nominal GDP growth rate of the economy.
  • WACC is the Weighted Average Cost of Capital, the discount rate I use in the DCF model.

Example: Let's use the Tech Innovators Inc. example from before:


FCFF_5 = $50 million
WACC = 8.21%
g = 2.5%

Using the formula, my calculation is:


Terminal Value = (50M * (1 + 0.025)) / (0.0821 - 0.025)
Terminal Value = 51.25M / 0.0571
Terminal Value = $897.55M

This gives me the value of the company at the end of Year 5. This value must still be discounted back to the present day to be used in my DCF model.

Method 2: The Exit Multiple Method

This is another method I use to estimate the Terminal Value, and it's often a good way to cross-check my first method. It involves applying a valuation multiple, like EV/EBITDA, to a company's final year's EBITDA. The idea is that a company will be valued at the end of the forecast period based on what similar companies are trading for today.

The formula is as follows:

Terminal\ Value = FCFF_n \times EV/EBITDA_{multiple}

Example: Continuing with Tech Innovators Inc.


Projected EBITDA for Year 5 = $75 million
Average EV/EBITDA multiple for similar companies = 10x

Using the formula, my calculation is:


Terminal Value = $75M * 10
Terminal Value = $750M

This value of $750M would then be discounted back to its present value and added to the present value of the forecasted cash flows to get the company's total intrinsic value.

Conclusion

Both methods are commonly used, and a robust DCF analysis will often calculate the Terminal Value using both methods and then average the results or use one to cross-check the other. It's a critical part of my valuation process, and understanding both methods helps me build a more accurate and reliable model.
